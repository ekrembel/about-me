CoinInRest


This is another version of Coin project, created with Angular and ASP.NET Core Web Api. In this application, users can do pretty much everything they can do in Coin.




Technologies used to build the application are

  • C#
  • ASP.Net Core Web Api
  • Angular
  • Postman
  • Identity Framework
  • JavaScript
  • HTML
  • CSS
  • Microsoft SQL Server
  • JSON
  • Visual Studio 2019
  • Bootstrap
  • Toastr
  • Open API
